Component: Dark / Light (Day / Night) Theme switching via system settings, time, or buttons

This component adds automatic theme switching (dark/light) based on the user’s OS preference, optionally by time of day, and optionally with UI buttons so the user can override the theme. The current mode is reflected on the <html> element via data attributes.

Overview

The Dark/Lite component detects the preferred theme and applies a corresponding attribute on <html>: data-anim-darklite-light or data-anim-darklite-dark. You can then style your UI using these attributes as selectors.

How to style
Use attribute selectors like: html[data-anim-darklite-dark] { ... } and html[data-anim-darklite-light] { ... }

Component location

Files are located in src/components/effects/darklite. Inside you’ll find:

  • JS file with theme detection and switching logic.
  • SCSS file for optional component styling.

Basic usage (system preference)

Add data-anim-darklite to your page (recommended on <html>). This is enough to detect the user’s OS preference and apply the correct mode attribute.

Enable dark/lite detection
<html data-anim-darklite>
...

After initialization, the component will add one of these attributes to <html>:

  • data-anim-darklite-light — light mode
  • data-anim-darklite-dark — dark mode

Time-based mode (day / night)

If you want the theme to switch depending on time of day, add data-anim-darklite-time.

Enable time-based switching
<html data-anim-darklite data-anim-darklite-time>
...

By default, data-anim-darklite-dark is applied from 18:00 to 05:00. All other time is considered light mode.

Custom time range

To set your own “night” time range, pass a value to data-anim-darklite-time as start,end (hours, 0–23).

Custom time window
<html data-anim-darklite data-anim-darklite-time="13,23">
...

In this example, dark mode is applied from 13:00 to 23:00.

User controls (toggle & reset)

If you want the user to switch the theme manually, add a button with data-anim-darklite-set. The component will toggle the theme and store the user’s choice in the browser.

Toggle theme button
<button type="button" data-anim-darklite-set>Toggle theme</button>

To let the user remove the saved preference (return to auto mode), add a reset button with data-anim-darklite-reset.

Reset stored choice
<button type="button" data-anim-darklite-reset>Reset theme</button>
Tip
Place the toggle in the header or settings panel. Use icons (sun/moon) and add accessible text for screen readers.
