Component: Cursor Custom mouse cursor with animation and shadow

The Cursor component allows you to replace the default system mouse cursor with a fully custom, animated version. You can style it freely and optionally add a smooth trailing shadow for a more “premium” interactive feel.

What is the Cursor component?

The Cursor component replaces the default browser cursor with a custom HTML-based cursor. This allows you to animate it, scale it, change shapes, or react to hover states in ways that are impossible with the native cursor.

Design tip
Custom cursors work best on creative websites, portfolios, landing pages, and interactive experiences. Avoid using them in admin panels or data-heavy UIs.

Component location

All component files are located in src/components/effects/cursor.

  • JS file with cursor logic and mouse tracking.
  • SCSS file for styling the cursor and its animations.

Basic usage

To enable the custom cursor, add the attribute data-anim-cursor to the <html> element. The component will initialize automatically.

Enable custom cursor
<html data-anim-cursor>
	...
</html>

Once enabled, the default system cursor will be hidden and replaced by the custom one defined in the component styles.

Cursor shadow (trail)

You can add a smooth trailing element (often called a “cursor shadow”) that follows the main cursor with a slight delay.

To enable the shadow, add the attribute data-anim-cursor-shadow alongside the main cursor attribute.

Enable cursor shadow
<html data-anim-cursor data-anim-cursor-shadow>
	...
</html>
Note
The shadow is purely decorative. You can control its size, opacity, delay, and animation entirely via SCSS.

Styling and customization

The Cursor component is intentionally minimal in logic. All visual appearance is controlled via SCSS.

  • Change size, shape, and color of the cursor.
  • Add scale or rotation on hover.
  • Animate transitions between states.
  • Disable the cursor on touch devices via media queries.
