Component: Marquee Animated running line with flexible direction

The Marquee component creates a continuously animated line of elements. It’s perfect for logos, announcements, tags, news tickers, or decorative UI motion. The component supports speed control, spacing, hover pause, and multiple directions.

What is a marquee?

A marquee is an animated container that continuously moves its child elements in a loop. Unlike the deprecated HTML <marquee> tag, this component is fully customizable, performant, and accessible.

Use cases
Brand logos, partner lists, promo messages, headlines, hashtags, testimonials, or subtle decorative motion.

Component location

The Marquee component files are located in: src/components/effects/marquee

  • JS file with animation logic.
  • HTML example snippet.
  • SCSS file for styling and layout.

Basic usage

To enable the marquee, wrap your elements in a container with the attribute data-anim-marquee. The JavaScript logic will connect automatically.

Basic marquee markup

<div data-anim-marquee>
	<div>Item one</div>
	<div>Item two</div>
	<div>Item three</div>
</div>

Spacing and speed

You can control the distance between elements and the animation speed using data attributes.

  • data-anim-marquee-space — spacing between items (px, default: 30)
  • data-anim-marquee-speed — animation speed in milliseconds (default: 1000)
Custom spacing & speed

<div data-anim-marquee data-anim-marquee-space="50" data-anim-marquee-speed="2000">
	<div>Item one</div>
	<div>Item two</div>
</div>

Direction and pause on hover

By default, the marquee moves from right to left. You can change the direction or pause the animation on hover.

  • data-anim-marquee-direction — left / right / top / bottom
  • data-anim-marquee-pause — pauses animation on hover
Direction & hover pause

<div data-anim-marquee
	data-anim-marquee-direction="right"
	data-anim-marquee-pause>
	<div>Item one</div>
	<div>Item two</div>
</div>
Vertical marquee
When using top or bottom direction, the container must have a fixed height and overflow: hidden.
