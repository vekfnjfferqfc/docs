Component: ScrollTo Smooth in-page navigation with optional active section highlighting

This component adds smooth scrolling to a target section after clicking a link/button. If the page URL contains a hash and a matching section exists on the page, scrolling happens automatically on load. Optionally, you can build a “scroll spy” navigation that highlights the current link while you scroll.

Basic usage

Add data-anim-scrollto to a navigation element and set its value to a CSS selector of the block you want to scroll to.

Scroll to a class or an id
<a href="#" data-anim-scrollto=".some-section" class="link">Go to section</a>
<a href="#" data-anim-scrollto="#contacts" class="link">Contacts</a>
Tip
You can use ScrollTo on any element (buttons, cards, etc.), not only on links.

Account for a fixed header

If you have a fixed header and you don’t want the scroll position to land “under” it, add data-anim-scrollto-header to the navigation element.

Scroll with header offset

<a
	href="#"
	data-anim-scrollto-header
	data-anim-scrollto=".some-section"
	class="link"
>Go to section</a>

Add an extra top offset

If you need an additional offset (e.g. spacing above the block), use data-anim-scrollto-top and provide a number (pixels).

Scroll with extra offset

<a
	href="#"
	data-anim-scrollto-top="30"
	data-anim-scrollto="#contacts"
	class="link"
>Contacts</a>

Highlight the active navigation item

To enable automatic “active item” highlighting while you scroll, you need the Watcher component. Add data-anim-watcher="navigator" to the sections you want to track.

ScrollTo + Watcher (navigator mode)

<a href="#" data-anim-scrollto=".some-section" class="link">Section</a>

...

<section data-anim-watcher="navigator" class="some-section">...</section>

When the tracked section is in view, the corresponding navigation link receives the class --navigator-active. When the section leaves the viewport, the class is removed.

Styling
Use .link.--navigator-active (or your selector) to style the active state.

Scroll by hash on page load

If a URL contains a hash and a matching section is found, ScrollTo will automatically scroll to it. The simplest approach is to use a class name in the hash.

Example URL + section

https://your-site.com/index.html#some-section

<section class="some-section"> ... </section>
Note
If you prefer, you can also target an #id — just make sure it matches the hash.
